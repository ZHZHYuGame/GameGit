语言Key表 Assets/Script/Language/LanguageInfo.cs

面板中语言绑定
1.在初始化时,绑定事件
private void Start()
{
	MessageEventMgr.GetInstance().AddListener(MessageType.TableChanged, SetText);
}
2.在面板销毁时,解绑事件
private void OnDestroy()
{
    MessageEventMgr.GetInstance().RemoveListener(MessageType.TableChanged, SetText);
}
3.事件方法
public void SetText(Message message)
{
    //消息转换
    StringTable value = message.data as StringTable;
    //文本赋值
    ChineseText.text = LanguageMgr.GetText(value, LanguageInfo.CHINESE_TEXT);
}
